# PRD: 큐(cuee)
 
## 한 줄 정의
큐는 고령 사용자가 다른 앱을 쓰다가 막혔을 때, 화면 위 동그라미를 눌러 말하면 다음에 눌러야 할 곳만 남기고 나머지를 하얗게 가려주는 Android 접근성 길잡이다.

## 제품 철학
- 빠르게 출시한다. 넓게 대충보다 좁게 90% 이상 성공하는 MVP를 만든다.
- 사용자가 직접 누른다. 큐는 대신 결제, 송금, 전송, 주문 확정을 하지 않는다.
- 신뢰가 핵심이다. 못 찾으면 솔직히 말하고, 민감 정보 화면에서는 멈춘다.
- 서버, 로그인, 서버 DB, AI API 없이 시작한다. 데이터는 기기 안에만 둔다.
- UI는 적게 만든다. 핵심 경험은 앱 내부 화면이 아니라 다른 앱 위 오버레이다.

## 대상 사용자
스마트폰 앱 사용 중 버튼 위치, 입력칸, 다음 행동을 찾기 어려운 고령 사용자. 초기 MVP는 보호자 설정 흐름을 따로 만들지 않고, 사용자가 직접 쓰는 흐름으로 설계한다.

## MVP 지원 앱과 명령
V1은 3개 앱, 6개 이하 명령만 지원한다.

### 쿠팡
- 상품 찾기: "물티슈 찾아줘", "마스크 사고 싶어"
- 장바구니 보기

### 카카오톡
- 메시지 입력: "아들한테 밥 먹었냐고 보내고 싶어"
- 사진/앨범 버튼 찾기

### 코레일톡
- 예매한 표 보기
- 승차권 예매 찾기

## 핵심 사용자 가치
- 버튼을 찾지 못해 가족에게 묻는 순간을 줄인다.
- 타이핑 부담을 줄인다. 쿠팡 검색어와 카톡 메시지는 자동 입력을 시도한다.
- 위험한 행동은 대신하지 않아 사용자와 가족의 불안을 줄인다.

## 핵심 작동 방식
1. 사용자는 지원 앱을 직접 연다.
2. 큐 동그라미가 화면 좌우 가장자리에 떠 있다.
3. 사용자가 동그라미를 누른다.
4. 큐가 "무슨 도움이 필요한지 말해보세요."라고 안내한다.
5. 사용자가 말한다.
6. 큐가 현재 화면의 접근성 노드를 분석한다.
7. 후보가 명확하면 필요한 영역만 남기고 나머지를 흰색 마스킹한다.
8. 사용자가 직접 누른다.
9. 큐는 이전 요청을 기억하고 최대 5단계까지 이어서 안내한다.

## 마스킹 UX
- 시각적으로는 전체 화면이 흰색으로 덮이고 눌러야 할 곳만 보인다.
- 구현은 터치 안정성을 위해 노출 영역을 제외한 주변 흰색 오버레이 조각으로 만든다.
- 마스킹 중 화면 텍스트는 표시하지 않는다.
- 음성으로만 짧게 "눌러주세요."라고 안내한다.
- 좌측 상단에 화살표 아이콘만 표시한다. 누르면 큐 안내만 닫힌다.
- 6초 무동작 시 한 번 더 음성 안내, 12초 무동작 시 자동 종료한다.

## 동그라미 UX
- Facebook Messenger chat head와 유사한 크기, 약 60dp.
- 사용자가 드래그 가능하다.
- 놓으면 가장 가까운 좌/우 가장자리에 부드럽게 붙는다.
- 상/하 가장자리에 붙지는 않는다.
- 드래그 중 하단 중앙 X 영역에 넣으면 동그라미가 꺼진다.
- 홈 화면에서 다시 켤 수 있다.

## 자동 입력 정책
허용:
- 쿠팡 검색어
- 카카오톡 메시지 본문

금지:
- 주소, 전화번호, 계좌번호, 카드번호, 비밀번호, 인증번호, 주민등록번호
- 결제, 송금, 주문 확정, 구매 확정, 보내기 버튼 안내

자동 입력은 `AccessibilityNodeInfo.ACTION_SET_TEXT`를 우선 사용한다. 실패 시 마이크 버튼 안내 또는 직접 입력 안내로 fallback한다.

## 안전 정책
- 결제/송금/주문확정/구매확정/인증/비밀번호/카드/계좌 관련 노드는 후보에서 제외한다.
- 민감 화면에서는 연속 안내를 중단한다.
- 민감 화면 문구: "개인정보가 있는 화면이에요. 직접 확인해 주세요."
- 필요 시 "입력칸 위치만 알려드릴까요?" 수준까지만 허용한다.

## 성공 기준
- 지원 앱 3개, 지원 명령 6개 이하 범위 안에서 30회 중 27회 이상 성공.
- STT 결과 수신 후 1초 이내에 안내 마스크 또는 실패/스크롤 안내를 표시.
- 결제, 송금, 주문 확정, 보내기 버튼을 안내하지 않음.
- 사용자가 직접 누를 수 있는 실제 영역만 노출.

## 비범위
- 서버 DB
- 로그인
- AI API 기반 의도 분석
- 화면/음성 원문 로그 저장
- 전체 앱 범용 자동화
- 자동 클릭
- 금융 송금/결제 완료 지원
- 보호자 계정/원격 관리
